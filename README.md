## Schedulr

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/schedulr.git
cd schedulr
go build -o schedulr .
```

### Usage

```bash
./schedulr --help
```

---

## Quick Start

```bash
./schedulr [command]
```
