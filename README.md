# SCHEDULR

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/schedulr.git
cd schedulr
go build -o schedulr .
```

## Usage

```bash
./schedulr --help
```
