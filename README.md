- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/schedulr.git
cd schedulr
go build -o schedulr .
./schedulr --help
```

## 🚀 Quick Start

```bash
./schedulr -h
```

## 🧪 Examples

```bash
# Example 1
./schedulr status
```
